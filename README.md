# 在CocoaPod中安装有米原生广告

在你的项目的根目录中新建一个 `Podfile` 文件，添加内容如下：

```
pod 'nativead'
```

终端命令行（使用 `cd` 命令）进入到您的iOS项目的根目录中，执行如下命令安装：

```
pod install
```

安装完成之后，点击目录中的.xcworkplace文件来打开项目，接下来你就可以在项目中使用有米原生广告了。
